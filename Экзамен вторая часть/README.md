## Подключение к базе

По умолчанию сервер использует:

```text
host: localhost
port: 5432
database: ekz
user: postgres
password: 123
```

Если параметры отличаются, создайте `.env` по примеру `.env.example`.

## Запуск

```bash
npm install
npm start
```

Откройте:

```text
http://localhost:3000
```

Пример пользователя из дампа:

```text
login: kondratieva@cake-shop.ru
password: AdCk76#
```
